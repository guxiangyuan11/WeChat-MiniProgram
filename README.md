# 微信小程序-文章电影
### 基于微信小程序开发的一款文章电影小程序，仅供学习
数据接口文章页数据接口是自己编写的，电影页面数据接口来自豆瓣API，
*******
以实现的功能有文章页，文章详情页，文章内部音乐播放，电影展示页，电影列表页，电影查询页，电影查询页
